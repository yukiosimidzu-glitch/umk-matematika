📖 Что такое корень n-й степени

Корнем n-й степени из числа a называется такое число b, которое при возведении в степень n даёт a. Записывается: , где .

  • n — показатель корня (n ≥ 2, целое число)
  • a — подкоренное выражение
  • При чётном n подкоренное выражение должно быть ≥ 0
  • При нечётном n подкоренное выражение может быть любым
