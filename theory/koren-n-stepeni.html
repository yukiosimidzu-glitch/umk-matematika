Раздел 2
📋 Свойства корня n-й степени Экзамен
📋 Пять основных свойств Выучить!

Все свойства справедливы для любых n ≥ 2. Для чётного n подкоренные выражения должны быть неотрицательными.

1. Корень произведения
2. Корень частного (b ≠ 0)
3. Возведение корня в степень n — это следует из определения корня
4. Корень из степени — можно менять порядок: сначала извлечь корень, потом возвести в степень, или наоборот
5. Корень из корня — показатели корней перемножаются
🔗 Как применять свойство 1 для упрощения

Если под корнем стоит произведение, и один из множителей — полный квадрат (или куб и т.д.), его можно вынести из-под корня. Это называется «вынесение множителя из-под знака корня».

🔢 Алгоритм решения Выучить!
Шаг 1Уединить корень. Перенести все члены без корня в одну сторону, оставив корень в другой. Если корней несколько — уединить один из них.
Шаг 2Возвести обе части в степень. Степень равна показателю корня. Если корень квадратный — возводим в квадрат, если кубический — в куб.
Шаг 3Решить получившееся уравнение. После возведения в степень корень исчезает, и получается обычное уравнение (линейное, квадратное и т.д.).
Шаг 4Обязательно сделать проверку! Подставить найденные корни в исходное уравнение. Возведение в квадрат может добавить посторонние корни.
