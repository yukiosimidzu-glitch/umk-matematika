Раздел 1
📐 Определение и правила нахождения производной Экзамен
📖 Что такое производная (простыми словами)

Производная функции в точке — это скорость изменения функции в этой точке. Она отвечает на вопрос: «Насколько быстро меняется y, когда x чуть-чуть увеличивается?»

Обозначение: — «эф штрих от икс». Также используют .

Главное правило: производная постоянного числа равна нулю. Производная x равна 1. Производная x² равна 2x.

Раздел 3
📈 Экстремумы функции (максимум и минимум) Экзамен
📖 Что такое экстремумы
  • Максимум — рост сменяется убыванием (↗ → ↘).
  • Минимум — убывание сменяется ростом (↘ → ↗).

Точки, где f'(x) = 0 — критические. Не каждая из них — экстремум.

Рис. 2 — График с максимумом и минимумом
[Подъём → вершина → спуск → впадина → подъём]
🔢 Алгоритм Выучить!
1.Найти f'(x)
2.Решить f'(x) = 0 → критические точки
3.Разбить прямую на интервалы
4.Проверить знак f'(x) на каждом интервале
5.+ → растёт ↗, − → убывает ↘
6.Смена +→− = max, −→+ = min, нет смены = нет экстремума
🏗️ Пример (забор 60 м у стены)

x < 15: S' > 0 ↗, x > 15: S' < 0 ↘ → максимум при x = 15.

Размеры: 15×30 м, площадь 450 м².

⚠️ Частая ошибка

. , но экстремума нет — знак не меняется. Всегда проверяйте смену знака.

📐 Нахождение производной

📏 Касательная

📈 Экстремумы

  • f'(x)=0 → критические точки
  • f'(x)>0 ↗, f'(x)<0 ↘
  • +→− = max, −→+ = min
  • Нет смены = нет экстремума

⚠️ Важно

  • f'(x)=0 не всегда экстремум
  • x³: f'(0)=0, но экстремума нет
  • Проверяйте смену знака
