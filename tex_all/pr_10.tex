\textbf{Задача 1.} Найдите производную и критические точки: $f(x) = x^2 - 6x + 5$\par\vspace{10pt}
\textbf{Задача 2.} Найдите промежутки возрастания и убывания: $f(x) = x^2 - 8x + 15$\par\vspace{10pt}
\textbf{Задача 3.} Найдите производную: $f(x) = 3x^4 - 2x^2 + 7$\par\vspace{10pt}
\textbf{Задача 4.} Найдите производную: $f(x) = (x + 2)(x - 3)$\par\vspace{10pt}
\textbf{Задача 5.} Исследуйте функцию на экстремумы: $f(x) = x^3 - 3x^2 + 2$\par\vspace{10pt}
\textbf{Задача 6.} Найдите наибольшее и наименьшее значение: $f(x) = x^2 - 4x + 3$ на $[0;\ 5]$\par\vspace{10pt}
\textbf{Задача 7.} Исследуйте функцию: $f(x) = x^3 - 6x^2 + 9x + 1$\par\vspace{10pt}
\textbf{Задача 8.} Число 12 разбейте на два слагаемых так, чтобы их произведение было наибольшим.\par\vspace{10pt}
\textbf{Задача 9.} Найдите наибольшее значение функции: $f(x) = -x^2 + 6x - 5$\par\vspace{10pt}
\textbf{Задача 10.} Площадка у стены, 80 м забора. Найти максимальную площадь.\par\vspace{10pt}
\newpage
\section*{Ответы}

\begin{enumerate}
  \item Ответ: $f'(x) = 2x - 6$; критическая точка $x = 3$.
  \item Ответ: убывает на $(-\infty;\ 4)$, возрастает на $(4;\ +\infty)$.
  \item Ответ: $f'(x) = 12x^3 - 4x$.
  \item Ответ: $f'(x) = 2x - 1$.
  \item Ответ: максимум $(0;\ 2)$, минимум $(2;\ -2)$.
  \item Ответ: $f\_{\min} = -1,\ f\_{\max} = 8$.
  \item Ответ: максимум $(1;\ 5)$, минимум $(3;\ 1)$.
  \item Ответ: 6 и 6, максимальное произведение = 36.
  \item Ответ: $f\_{\max} = 4$.
  \item Ответ: 20 м $\times$ 40 м, $S\_{\max} = 800$ м².
\end{enumerate}
