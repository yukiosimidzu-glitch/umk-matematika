\textbf{Задача 1.} Упростите: $\sin^2 \alpha + \cos^2 \alpha$\par\vspace{10pt}
\textbf{Задача 2.} Упростите: $\tg \alpha \cdot \ctg \alpha$\par\vspace{10pt}
\textbf{Задача 3.} Найдите $\cos \alpha$, если $\sin \alpha = \dfrac{3}{5}$ и $0^\circ < \alpha < 90^\circ$.\par\vspace{10pt}
\textbf{Задача 4.} Найдите $\tg \alpha$, если $\sin \alpha = 0{,}6,\ \cos \alpha = 0{,}8$.\par\vspace{10pt}
\textbf{Задача 5.} Упростите: $(\sin \alpha + \cos \alpha)^2 - 2\sin \alpha \cos \alpha$\par\vspace{10pt}
\textbf{Задача 6.} Найдите $\sin \alpha$, если $\cos \alpha = -\dfrac{5}{13}$ и $90^\circ < \alpha < 180^\circ$.\par\vspace{10pt}
\textbf{Задача 7.} Докажите тождество: $\tg \alpha + \ctg \alpha = \dfrac{1}{\sin \alpha \cos \alpha}$\par\vspace{10pt}
\textbf{Задача 8.} Упростите: $(1 - \sin \alpha)(1 + \sin \alpha)$\par\vspace{10pt}
\textbf{Задача 9.} Зная, что $\tg \alpha = 2$, найдите: $\dfrac{\sin \alpha + \cos \alpha}{\sin \alpha - \cos \alpha}$\par\vspace{10pt}
\textbf{Задача 10.} Уклон кровли задан углом $\alpha$, для которого $\sin \alpha = 0{,}6$. Найдите $\tg \alpha$ — отношение высоты подъёма к горизонтальной проекции ската.\par\vspace{10pt}
