Решите уравнение: sin x = 0

sin x = 0 в точках пересечения единичной окружности с осью Ox.

Ответ: x = πk, k ∈ Z.

Стены Задача 6 (2 балла)

Стены Задача 7 (2 балла)

Шаг 1. tg x = 1.

tg x = 1 при x = π4. Период тангенса — π.

Шаг 2. x = π4 + πk, k ∈ Z.

Ответ: x = π4 + πk, k ∈ Z.

Кровля Задача 8 (3 балла)

Решите уравнение: 2 cos² x – 1 = 0

Шаг 1. 2 cos² x = 1 ⇒ cos² x = 12

Шаг 2. cos x = ± √1/2 = ± √22

Шаг 3. cos x = √22 ⇒ x = ± π4 + 2πk.

cos x = –√22 ⇒ x = ± 3π4 + 2πk.

Объединяем: x = π4 + πk2, k ∈ Z.

Ответ: x = π4 + πk2, k ∈ Z.

Кровля Задача 9 (3 балла)

Решите уравнение на [0; 2π]: sin x = –12

sin x = –12. Базовый угол: π6. Синус отрицателен в III и IV четвертях.

III четверть: x = π + π6 = 7π6.

IV четверть: x = 2π – π6 = 11π6.

Оба значения ∈ [0; 2π].

Ответ: 7π6; 11π6.

Кровля Задача 10 (3 балла) 🏗️

Угол наклона кровли α удовлетворяет уравнению tg α = 0,5. Найдите α в градусах (округлите до целых).

tg α = 0,5 — не табличное значение.

α = arctg(0,5) ≈ 26,57°.

Округляем до целых: ≈ 27°.

Ответ: α ≈ 27°.
