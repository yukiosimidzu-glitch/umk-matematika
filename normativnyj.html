📄 Образовательный стандарт

Выписка из образовательного стандарта общего среднего образования по учебному предмету «Математика» (базовый уровень).

📌 Знания
  • Основные понятия алгебры и геометрии для решения прикладных задач.
  • Свойства степеней, корней, логарифмов, тригонометрических функций.
  • Методы решения уравнений, неравенств, систем.
  • Понятие производной, её геометрический и физический смысл.
  • Основные понятия стереометрии: многогранники, тела вращения.
📌 Умения
  • Выполнять тождественные преобразования выражений.
  • Решать рациональные, иррациональные, показательные, логарифмические и тригонометрические уравнения.
  • Исследовать функции с помощью производной.
  • Вычислять площади поверхностей и объёмы геометрических тел.

🎯 Формируемые компетенции

🔹 Профессиональная
  • Применение математических методов для расчёта строительных конструкций.
  • Выполнение геометрических построений, работа с чертежами и сечениями.
  • Расчёт объёмов, площадей, углов и уклонов в профессиональных задачах.
🔹 Информационная
  • Работа с цифровыми инструментами (GeoGebra, Desmos).
  • Использование онлайн-ресурсов для самопроверки и построения графиков.
  • Навыки работы с QR-кодами для доступа к учебным материалам.
🔹 Коммуникативная
  • Оформление результатов вычислений в виде таблиц, схем и чертежей.
  • Умение аргументировать выбор метода решения.
  • Работа в группе при решении практико-ориентированных задач.
